The "content" plugin

The "content" plugin is a text editing field. As it makes its handle contentEditable, it offers a perfect integration with the page to create a WYSIWYG experience. You can use it to edit text that flows around floating elements.

Parameters

  • noedit (true|false) : set it to true to make the field read only (constant).
  • class (name) : set a class name on the handle
