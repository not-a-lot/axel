The "link" plugin

The "link" plugin is a dual text entry field to edit the anchor text and the target part (URL) of a link. It's static view shows only the anchor text. Once clicked, it popups a window lens around the anchor text with the two editing fields.

While editing it is possible to navigate from the anchor text input to the URL input using key up and key down. Future versions should go farther and use standard tab navigation instead to fully integrate with the page.

This plugin is only compatible with a xt:use element.

Parameters

  • linkRefTagName (linkRef) : string label used by the load/save method for the url content
  • linkTextTagName (linkText) : string label used by the load/save method for the anchor text content
  • trigger (mouseover|click) : user action that triggers editing mode
