The "video" plugin

The "video" plugin is a video URL entry field. The video is shown inside an embedded video player. When the mouse is passed over it displays a popup window lens around the player that allows to change the video URL.

Currently this plugin is limited to You Tube video URLs. There is also a limitation in some browsers that do not allows to play the video (the mask used to detect mouse over filters out events).

Parameters

  • lang (fr_FR) : lang's code for player's GUI (must respect the video's provider's standards)
  • inputFieldMessage (Paste the video's link here) : text appearing in the input field when the model does not contain a valid URL
  • width (number) : width of the embedded video player
  • height (number) : height of the flash video player
